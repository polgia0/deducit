View Data Set


The menu allows the exploration of all the Data Set. Several commands facilitate the task. At the top, the Variable Names are listed with a check box, If one of the thick is removed, the Variable is removed from the following view (it is not removed from the Data Set).
This is helpful when many Variables are present and only few of them are relevant for the view. In the follwing example all the variables are shown since the Data Set is very thin.
The "Show  entries" combo box on the right , states the number of rows in the next table . Some fix values are allowed (10,25,50 and 100)  and must be defined depending of the Data Set dimension. The bottom line of the table indicates how many object (entries) are displayed and how many are in total. In the following example 1 to 10 of 150 entries are shown. To see the the next tens, the right buttons 1,2,3,4,5,...15, need to be pushed.
The data in the table are listed as in the original Data Set. The name of each Variable is shown at the top of each column. Each raw represents a different object. On the right of each Variable Name a widget allows to order the column from the minimum to the maximum or vice versa. When this action is made, all the table is reordered following the rule based on the selected variable. The ordering is done only for the view purpose, data in the Data Set are unchanged. The ordering criteria clearly is valid only for numeric variables, however it may work for text variable too, but results are quite unpredictable.
At the top right also a Search input box is present. It allows the research of a specific value in the data. Just dial in the box and hit return. The usefulness of this search is evident for very big Data Set and only when few and defined values are required.



It is important to note there are two tables in the view menu. The top one is for the objects in the Training Data Set, the bottom one is for the Test objects.
