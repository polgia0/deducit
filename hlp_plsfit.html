PLS Fit and Prediction


  The Prediction of a PLS model is quite straightforward as soon as the model parameters are defined. The prediction is always cross-validated unless the Validation option is set to None in  Setting Menu. The prediction data are visualized as shown in the figure for the gasoline Data Set. A square plot graph Predicted vs Measured values is displayed. The PLS prediction is as good as data are on the plot diagonal, i.e. coincidence between predicted and measured numbers.



Of course, there is a plot for each Response and the plot changes depending of the top Combo selector at the top. Another interesting possibility is to see the spreading that the Cross Validation has evaluated. Selecting the Check Box of CV Spread, the prediction is repeated several times on random bases and the predictions of each iteration is averaged. In the case of the example the Response is shown in the next figure.



Now, points are replaced with error bars. If the bars cross the diagonal the prediction is reliable, if not, the prediction retains some model error. Objects with larger error bars are the most interesting since are the outlier of our regression.
