Training/Test Selection


The Training/Test menu allows to split the Data Set rows in two Groups: the first made by the objects use to set up models as Training Set; the second made by the object used to tests the models when they are formulated. The menu is used together with the menus: "Additional Data" that will be described later. The box on the left (Training Objects) lists the Row Index of each object in the Data Set. As it is done in the object validation, the object that is considered a test need to be highlighted first. Then, pushing the right hand arrow, its row index is moved from the left to the right box, i.e.the box of Test Objects. Selection and transfer can be done using single or multiple objects and can be moved in both sides. The objects in the right side, i.e.in the Test list, will not be used in the model construction such us PCA and PLS.
