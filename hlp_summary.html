Summary Report


The summary report is a table that highlights some useful information about the working Data Set. There are:  the number of elements (including those missing), the number of positive, negative and null elements, the number of missing (empty) elements, the number of columns and rows. At the end the split between the number of Training and Test Set elements. All data are based on the original Data Set and they are not affected by user choices, except the number of Training/Test split.
